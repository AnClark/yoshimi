PadSynth

The PadSynth engine has two tabbed views in a single window. You will normally be shown the harmonic structure, the envelopes one (as it's name suggests) consisting mostly of envelope and LFO inserts.

There is a button to give access to an overall Resonance that can be applied to the engine.

Below this is a Waveform button which gives access to a highly detailed waveshape editor.
