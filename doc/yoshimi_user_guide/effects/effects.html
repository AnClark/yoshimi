Effects

Effects can be applied to single instruments or as master effects.
For any effect, if you select a preset and then change parameters this dropdown will become blue to remind you the preset has changed.
Note that for some effects, the D/W (Dry/Wet) parameter when editing the effect for a single instrument becomes a Vol (Volume) parameter when applied as Master effects. These are controlled from the Main window.

  • R.S.: Room Size (%)
  • D/W: Dry/Wet balance. If completely to the left only the dry sinal (no reverb) will be heard
  • Pan: The left/right panning (panorama) for the effect (% left/center/right)
  • Time: Reverberation time (in seconds)
  • I.del: Initial delay (also known as pre-delay) (milliseconds)
  • I.delfb: Initial delay feedback (%)
  • BW Bandwidth (cents, only available for type bandwidth)
  • E/R: Early reflections ?? (never seems to be active though...)
  • LPF: Low Pass Filter. This will cut frequencies above the selected value (Hertz)
  • HPF: High Pass Filter. This will cut frequencies below the selected value (Hertz)
  • Damp: Amount of damping of low frequencies (%)
